🎓 CareerCompass AI

CareerCompass AI is an AI-powered career guidance platform designed to help students plan their academic journey, develop technical skills, build industry-ready portfolios, and prepare for placements through personalized AI recommendations.

The platform combines Artificial Intelligence, analytics, and personalized learning to provide students with a complete career development experience.

---

 🚀 Features

- 👤 Student Digital Twin
- 🤖 AI Career Mentor
- 📚 Personalized Learning Coach
- 💡 AI Project Mentor
- 🛣️ Career Roadmap Generator
- 📄 AI Resume Analyzer
- 📈 AI Skill Gap Analyzer
- 📊 Analytics Dashboard
- 🎯 Career Goal Planning
- 📚 Academic Progress Tracking
- 💻 Technical Skills Management
- 🔗 GitHub & LinkedIn Integration
- 📅 Daily Study Target Tracking
- 🔥 Firebase Authentication & Database
- 📱 Fully Responsive User Interface

---

📋 Modules

🏠 Dashboard
Displays an overview of the student's academic progress, career goals, study targets, skills, and quick access to all important features.

---

👤 Student Digital Twin
Creates a personalized digital profile by storing academic information, technical skills, interests, career goals, and learning preferences to provide AI-powered recommendations.

---

🤖 AI Mentor Hub
Acts as an intelligent career mentor that provides:
- Career guidance
- Placement preparation tips
- Interview preparation
- Learning recommendations
- Career-related Q&A

---

📚 Learning Coach
Generates personalized learning plans by recommending:
- Courses
- Certifications
- Learning resources
- Daily study plans
- Skill improvement suggestions

---

💡 Project Mentor
Helps students build industry-ready projects by providing:
- Project ideas
- Technology recommendations
- Development guidance
- Best practices
- Project documentation support

---

🛣️ Roadmap Generator
Creates step-by-step career roadmaps for different domains such as:
- Software Engineering
- Artificial Intelligence
- Full Stack Development
- Data Science
- Cloud Computing
- Cyber Security

---

📄 Resume Analyzer
Analyzes resumes using AI and provides:
- Resume score
- Missing skills
- ATS optimization suggestions
- Resume improvement tips
- Placement readiness feedback

---

📈 AI Skill Gap Analyzer
Compares the student's current skills with industry requirements and identifies:
- Missing skills
- Recommended technologies
- Learning priorities
- Personalized improvement suggestions

---

📊 Analytics Dashboard
Provides visual insights including:
- CGPA Tracking
- Study Progress
- Skill Development
- Learning Statistics
- Career Readiness Score
- Performance Analytics

---

⚙️ Settings
Allows users to:
- Update profile information
- Manage academic details
- Change preferences
- Configure account settings
